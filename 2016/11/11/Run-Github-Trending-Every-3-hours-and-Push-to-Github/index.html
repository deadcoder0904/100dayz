So what is a Cron Job ?

Cron is a daemon, viz, a background process which runs at a specified time. If we want to take a backup daily then instead of doing it manually every fucking day, we write a script & provide it to cron, then cron runs the script in that specified time. Anacron is the cron for desktops and laptops. It does not expect the system to be running 24 x 7 like a server.

Syntax of crontab (field description)

The syntax is:

1 2 3 4 5 /path/to/command arg1 arg2

OR

1 2 3 4 5 /root/email.sh

Where,

1: Minute (0-59)

2: Hours (0-23)

3: Day (0-31)

4: Month (0-12 [12 == December])

5: Day of the week(0-7 [7 or 0 == sunday])
